## 脚本一
```shell
awk -F ":" '{print $2":"$3}' nowcoder.txt | sort | uniq -c | sort -nr | sed 's/[ \t]*//'
```


## 脚本二
```shell
awk -F ":" '{map[$2":"$3]++} END{for(k in map) print map[k],k}' nowcoder.txt | sort -nr
```
